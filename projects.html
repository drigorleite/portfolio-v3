⭐ Featured Project · Full-Stack SaaS

FlipWise

A full-featured SaaS platform for real estate investors and general contractors managing house-flipping and renovation projects. FlipWise centralises property tracking, expense management, worker timesheets, payroll, and financial reporting in a single, role-based platform — with AI-powered receipt OCR, Stripe billing, and PDF export.

Key Features

  • Role-based access control (Admin vs. Worker) with JWT authentication
  • AI-powered receipt OCR — upload a photo, get auto-categorised expenses
  • Construction timeline with visual Gantt-style progress tracking
  • Stripe subscription billing (Free / Pro / Business tiers)
  • Financial reports with PDF & CSV export per property
  • PWA-ready with mobile bottom nav and offline support
  • 75 automated tests passing (Vitest)
React TypeScript Node.js tRPC Drizzle ORM MySQL Stripe AWS S3 Tailwind CSS Vitest
